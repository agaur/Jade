jQuery UI - Tool tip plugin

Code 
$("#simpleToolTip").tooltip({
	'content':'A simple Tooltip',
});
 Hover Over me for a simple tooltip
Code 
$("#titleToolTip").tooltip({
	'title':'Title',
	'content':'A tooltip with title'
});
 Hover Over me to see a tooltip with title
Code 
$("#"speedToolTip"").tooltip({
	'content':'A tooltip with slow fadeout',
	'speed': 3000
});
 Hover Over me to see the toolTip with speed decreased
Code 
$("#customClassToolTip").tooltip({
	'title':'Title',
	'content':'A tooltip with custom class',
	'toolTipClass':'customToolTipClass'
});

.customToolTipClass {
	background:green;
}
.customToolTipClass-title{
	font-style:italic;
	font-size:xx-large;
	color:blue;
}
.customToolTipClass-content{
	font-family:cursive;
}
Hover over me to see a tooltip with custom class
Code 
$("#effectTooltip").tooltip({
	'content':'A tooltip with effect',
	'effect':'size'
});
Hover Over me to see effect in tooltip
